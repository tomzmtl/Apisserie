# Spark

## Yet another React boilerplate

Spark is an opinionated WebApp boilerplate.

### Core components

* **React** UI library
* **Redux** State management
* **Mocha** Test framework
* **Chai** Assertion library

### Additional integration

* Travis CI
* Heroku
